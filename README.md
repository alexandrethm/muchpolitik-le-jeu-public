# muchpolitik-le-jeu
 
Quelques explications.. Ceci est le repo GitHub du jeu où se trouve toutes les données du jeu importantes, et le plus à jour possible. Il est privé.

## Structure
Dans le répertoire ```core/src/``` se trouve **le code du** jeu, il est surtout utile au développeur.

Dans le répertoire ```android/assets/``` se trouve toutes les **ressources graphiques, audio, et données du jeu (cartes, dialogues)**. On y retrouve :
- Les niveaux dans le répertoire ```android/assets/data/levels/```
- Les dialogues dans le répertoire ```android/assets/data/dialogs/```

Si vous avez des modifications à faire dessus, merci de vous relire avant de poster, et de ne modifier que les fichiers nécessaires (pour limiter les erreurs).

## Comment faire
1. Télécharger GitHub Desktop (ou équivalent)
2. Aller voir sur google ou suivre les petits guides expliquant comment GitHub marche
3. Abracadabra plus de problèmes de versions différentes chez chacun
